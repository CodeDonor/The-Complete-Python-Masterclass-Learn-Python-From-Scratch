This is a complimentary section to Django which will teach you the basic concepts of bootstrap using which you will be able to design the front end for your Django application.

Bootstrap, although not related to Python helps in building the font end of our web apps in an efficient way and makes the web app look more attractive.

We have made use of Bootstrap 3 in the Django section, however I have uploaded the lectures for Bootstrap 4 as that is the latest version.

There are a few minor syntax changes between the two, however you will be able to catch up with them.

Although this section is not so extensive but it covers all the major parts related to bootstrap and should give you an idea of what bootstrap is and how it works.

I hope you find these lectures useful !
